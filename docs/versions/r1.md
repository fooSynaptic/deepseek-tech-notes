> [← 中文导读](../README.md) · [← 仓库首页（EN）](../../README.md) · [← 演进总览 §3.4](../reports/deepseek-version-lineage-20260625.md#34-deepseek-r1) · [← 版本目录](./README.md) · [RLVR 详解](./rlvr.md) · [Raschka §RLVR/GRPO](../reports/raschka-technical-deepseek-v3-v32-highlights.md#rlvr--grpo-要点326)

## 核心结论摘要

- R1（2025-01）基于 **V3-Base**，**架构与 V3 完全相同**（671B / 37B）。
- 核心差异在 **RLVR + GRPO** 后训练：可验证奖励替代神经 RM，无 critic。
- R1-Zero 为纯 RL 探索；R1 为 SFT → RL → 拒绝采样 SFT → 二阶段 RL。
- 推理 infra 与 V3 一致（MLA latent KV、vLLM 适配等）。

---

## 定位

2025 年 1 月发布，基于 **DeepSeek-V3-Base** 的 **推理专精** 模型。**架构与 V3 完全相同**（671B / 37B activated / 128K）；差异在 **[RLVR](./rlvr.md) + GRPO** 后训练，带火「可验证奖励强化学习」路线。

## 核心训练

| 组件 | 要点 |
|------|------|
| **基座** | DeepSeek-V3-Base，**零架构改动** |
| **RLVR** | 规则/验证器奖励，无神经 RM → [RLVR](./rlvr.md) |
| **GRPO** | 组内相对 advantage，无 critic | [RLVR §算法：GRPO](./rlvr.md#grpo) |
| **R1-Zero** | 纯 RL 探索路径（无 SFT 冷启动） |
| **R1** | 冷启动 SFT → RL → 拒绝采样 SFT → 二阶段 RL（+ 通用 RM） |

**训练 pipeline**：[DeepSeek-R1 训练 Pipeline](../material/papers/deepseek-r1/training-pipeline.md)

## 与相邻版本

| 维度 | V3 | R1 | V3.1 |
|------|----|----|------|
| 架构 | MoE + MLA | **同 V3** | **同 V3** |
| 定位 | 通用 Base | **推理 / CoT** | Hybrid chat + thinking |
| 后训练 | SFT + RL（V3 论文） | **RLVR 为主** | 另一套 post-train |
